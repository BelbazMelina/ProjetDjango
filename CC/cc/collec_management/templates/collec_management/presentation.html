Présentation de notre projet

Bienvenue dans l'application de gestion des collections.


Notre application vous permet de créer, visualiser et gérer vos collections (supprimer, modifier) de manière rapide et facile.

Vous pouvez consulter toutes les collections, il vous suffit d'ajouter à votre URL : /all/ dans votre navigateur.

Vous pouvez consulter une collection spécifique en utilisant son ID, il vous suffit d'ajouter à votre URL : /collection/(id)/ dans votre navigateur.

Pour créer une nouvelle collection, il vous suffit d'ajouter à votre URL : /new/ dans votre navigateur.

Pour modifier une collection existante, il vous suffit d'ajouter à votre URL : /edit/(id)/ dans votre navigateur.

Pour supprimer une collection, il vous suffit d'ajouter à votre URL : /delete/(id)/ dans votre navigateur.
